# Equity Metrics Dashboard: Maternal Health Disparities

## Project Goal
Analyze maternal health disparities to demonstrate the operational equity gap—where policy coverage exists but outcomes remain inequitable.

## Status
Week 1 - Initial Setup

## Data Source
- CDC WONDER: Maternal Mortality Data (2018-2022)
- CDC WONDER: Natality/Birth Data (2018-2022)

## Technologies
- Python 3.x
- Pandas (data analysis)
- Matplotlib & Seaborn (visualization)
- Jupyter Notebooks

## Project Structure
```
equity-metrics-dashboard/
├── data/
│   ├── raw/          # Original downloaded data
│   └── processed/    # Cleaned data
├── notebooks/        # Jupyter notebooks
├── src/              # Python modules
├── outputs/          # Charts and reports
└── README.md
```
